Markdown# Basic Calculator

A simple, fully functional calculator built with **HTML**, **CSS**, and **vanilla JavaScript**. This project demonstrates basic arithmetic operations: addition, subtraction, multiplication, and division.

This repository was created as a demo project for practicing **Modern Code Review (MCR)** process, including creating branches, submitting Pull Requests, peer reviewing code, addressing feedback, and merging changes.

## Features

- Perform basic operations: `+`, `-`, `×`, `÷`
- Decimal point support
- Clear (C) and Delete (DEL) functionality
- Responsive button layout
- Clean and modern user interface
- Error handling for invalid expressions

## Live Demo

Open `index.html` in any modern browser to use the calculator.
